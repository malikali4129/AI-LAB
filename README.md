# Fake AI Lab

A humorous AI parody website built with plain HTML, CSS, and JavaScript.

## Features

- Futuristic dark UI with glassmorphism and neon glow
- Animated home hero with typing and glitch effects
- A separate birthday calculator page linked from the homepage
- Responsive layout for mobile and desktop

## Run

Open `index.html` in a browser or use any static file server.

The birthday calculator lives on `birthday.html`.

## Notes

- No frameworks were used.
- Google Fonts are loaded from the browser at runtime.
- Additional modules can be added under `modules/` one at a time.
